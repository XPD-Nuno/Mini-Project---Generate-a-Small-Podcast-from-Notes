For this lab, I followed Path 1 and refactored my existing news podcast generator project. I added an APIResponse structure and an OpenAIWrapper class based on the refactoring lesson so that OpenAI API calls are handled in a cleaner and more reusable way. I also created helper functions to validate the news input and build the podcast prompt, which separated validation, prompt creation, and API logic into smaller focused pieces. The main challenge was making sure the API key was loaded before creating the wrapper client, and then ensuring invalid input stopped the workflow before audio generation. I learned how helper functions and API wrappers make the project easier to maintain, test, and debug.
